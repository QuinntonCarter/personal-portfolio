Whatchumeme?

A web based application in which users can create a profile, view their Spotify listening stats between 3 varying timeframe selections, post 1 recent playlist they've been enjoying as well as 1 piece of listening stats to be viewed by friends, and view listening stats plus playlists shared by friends. This project demonstrates my understanding of RESTFUL API structures and one to one, one user to one posted "mood" and one shared playlist, as well as many to many, user can friend many people and many people can friend the user, database relationships in NoSQL databases.
